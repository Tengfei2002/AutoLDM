1. 沟道数量参数 (Channel Count)
num_channel: 全局沟道（如纳米片或纳米线）数量定义。
num_channel_n / num_channel_p: 独立定义 N 型和 P 型器件的沟道数量。仅在 num_channel = -1 时生效。

2. 结构排布与单元尺寸 (Arrangement & Cell Metrics)
arrangement: CFET 器件的层间排布方式，用于决定上下层器件是对称排布（symmetrical）还是平行排布（parallel）。
cell_height: 芯片标准单元的整体高度。
cpp: 接触栅间距（Contacted Poly Pitch），即相邻两个栅极中心的物理距离。

3. 介电层与侧墙厚度 (Dielectrics & Spacers)
inner_spacer_thickness: 内侧墙厚度。用于在物理上隔离栅极与源漏外延区，主要作用是降低寄生电容。
gate_spacer_thickness: 栅极侧墙的厚度。
high_k_thickness: 高 K 栅介电层的厚度。
mdi_thickness: 相距最近的 NFET 和 PFET channel之间的距离
channel_thickness:沟道的厚度
channel_mdi_thickness:沟道之间的厚度

4. 栅极尺寸 (Gate Dimensions)
gate_length / gate_width: 全局统一的栅极长度和栅极宽度。
gate_upper_length / gate_lower_length: 独立定义上层和下层器件的栅极长度。仅在 gate_length = -1 时生效。
gate_upper_width / gate_lower_width: 独立定义上层和下层器件的栅极宽度。仅在 gate_width = -1 时生效。

5. 沟道尺寸与位置 (Channel Dimensions & Position)
channel_center_y: 沟道中心在 Y 轴（垂直方向）上的绝对坐标位置。
channel_width / channel_length: 全局统一的沟道宽度和长度。由于在结构设定中沟道与栅极方向垂直，沟道的实际物理长度通常由栅宽、高 K 层厚度以及内侧墙厚度共同决定。
channel_upper_length / channel_lower_length: 独立定义上层和下层器件的沟道长度。仅在 channel_length = -1 时生效。
channel_upper_width / channel_lower_width: 独立定义上层和下层器件的沟道宽度。仅在 channel_width = -1 时生效。

6. 源漏外延生长尺寸 (Source/Drain Epitaxial Overgrowth)
sd_overgrowth_x: 全局统一的源漏区在 X 方向（横向）的外延生长延伸量。
sd_overgrowth_y_up / sd_overgrowth_y_down: 全局统一的源漏区在 Y 方向（纵向）上，向上和向下两侧的外延生长延伸量。
sd_upper_overgrowth_x / sd_lower_overgrowth_x: 独立定义上层和下层器件源漏区在 X 方向上的外延生长量。仅在 sd_overgrowth_x = -1 时生效。
sd_upper_overgrowth_y_up / sd_upper_overgrowth_y_down: 独立定义上层器件源漏区在 Y 方向向上和向下的外延生长量。仅在 sd_overgrowth_y_up / _y_down = -1 时生效。
sd_lower_overgrowth_y_up / sd_lower_overgrowth_y_down: 独立定义下层器件源漏区在 Y 方向向上和向下的外延生长量。仅在 sd_overgrowth_y_up / _y_down = -1 时生效。
