# AutoLDM

AutoLDM 是面向 CFET/stacked nanosheet 结构的 layout-to-device 建模项目。当前主线流程是：

```text
layout txt + layer rules + architecture parameters
        |
        v
AutoLDM/LD/gen_sde.py
        |
        v
AutoLDM/LD/gen_sde.cmd
```

`gen_sde.cmd` 可在 Sentaurus Structure Editor 中执行，用于生成器件几何、contact、doping 和 mesh 设置。

## 快速运行

进入 `AutoLDM/LD`：

```powershell
cd C:\Users\Tengfei\Desktop\Project_DTCO\AutoLDM\LD
.\run_gen.bat
```

`run_gen.bat` 中通过无后缀文件名选择输入输出：

```bat
set "LAYOUT_NAME=test1"
set "ARCH_NAME=cfet"
set "LAYER_RULE_NAME=layer_rule_1"
set "OUTPUT_NAME=%LAYOUT_NAME%"
```

默认读取：

```text
gds/test1_gds.txt
rules/cfet_arch.txt
rules/layer_rule_1.txt
```

默认输出：

```text
SDE/test1.cmd
```

也可以指定输入输出：

```powershell
python gen_sde.py <layout_file> <arch_file> <layer_rule_file> <output_cmd_file>
```
## 输入文件

`gds/test1_gds.txt` 定义二维 layout 矩形：

```text
x1 y1 x2 y2 layer_num
```

`rules/layer_rule_1.txt` 定义 layout layer 的三维规则：

```text
layer_num layer_name enable height start_z1 end_z2 material boundary ILD
```

`rules/cfet_arch.txt` 定义结构参数，包括 channel 数量/尺寸、High-k 厚度、inner spacer、S/D 外延、doping 和 meshing。

## 当前结构关系

当前版本按图示关系理解各结构：

- `Gate`：白色核心区域，几何边界来自 layout gate 层。
- `High-k`：红色薄层，紧贴并包覆 gate 外表面。
- `Channel`：青绿色小块，位于 gate 左右两侧，每层拆成 `*_L` 和 `*_R`。
- `Inner_spacer`：棕黄色区域，与 channel 位于同一侧向 slab；channel 占据窗口，inner spacer 填充窗口之外的 y/z 间隔。
- `S/D`：与左右 channel 小块的外端面接触。

衬底不进行掺杂。S/D 默认掺杂为：

- 上层 N 型 S/D：`ArsenicActiveConcentration = 8e19`
- 下层 P 型 S/D：`BoronActiveConcentration = 8e19`

最终 mesh 输出名使用 `n@node@_` 前缀，例如：

```text
n@node@_cfet_structure
```
