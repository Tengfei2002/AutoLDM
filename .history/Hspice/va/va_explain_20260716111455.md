```shell
parameter real L=1.6e-8;
parameter real W=3.5e-8;
parameter real NF=1;

parameter real U0 = 3.0e-2;
parameter real XL=1.2e-8;		
parameter real DVTSHIFT=0.0;
parameter real DeltaWGAA = 0;
parameter real DeltaTGAA = 0;
parameter real EOT_0 = 1.1e-9;
```

这里的 W 被赋给了 WGAA：
```shell
real WGAA=W;
```

所以 W 在这个模型里表示 GAA/nanosheet 的 sheet width，默认 35 nm。模型开头也写了训练范围：
GAA global model (Lg: 16-100 nm; WGAA: 10-40 nm)

等效栅长：
```shell
Weff_UFCM = 2 * (WGAA + DeltaWGAA + TGAA + DeltaTGAA) * NGAA;
```
默认参数：
```shell
W = WGAA = 35 nm
TGAA = 5 nm
NGAA = 3
Weff ≈ 2 × (35 + 5) × 3 = 240 nm
```

NGAA 是纳米片个数
