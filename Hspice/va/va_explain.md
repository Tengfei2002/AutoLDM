# PHIMO-Fusion VA 校准说明

适用文件：`nmos_lvt_2.va` 与 `pmos_lvt_2.va`。它们分别由 `fusion_ic_nmos_lvt.va`、`fusion_ic_pmos_lvt.va` 完整复制后修改；默认数值未改变，因此**不给实例传入新参数时，模型的数学结果应与源文件一致**。

这不是把所有 `real` 都开放。PHIMO-Fusion 的短沟道、迁移率、量子和 C-V 修正中有神经网络输出；权重、归一化边界、网络输出中间量必须保持不变。这里开放的是网络以外的器件几何、材料边界条件、输运/输出特性和寄生电容项。

## 已开放为 `parameter real` 的量

### 原有实例与基本工艺参数

| 参数 | 默认值 / 单位 | 物理含义与首选数据 |
|---|---:|---|
| `L`, `W`, `NF` | 16 nm, 35 nm, 1 | 实例的栅长、片宽（`W` 初始化 `WGAA`）与并联指数量。它们是器件几何，不是同一节点全局拟合常数。 |
| `U0` | 0.03 m²/(V·s) | 低场迁移率基准；用长沟道、低 `VDS` 的 `Id–Vg` 线性区拟合。网络输出仍会对它施加几何相关修正。 |
| `XL` | 12 nm | 有效栅长偏移，`Lg=L+XL`；用多个栅长的 `Id–Vg/Id–Vd` 共同拟合。 |
| `DVTSHIFT` | 0 V | 全局阈值电压平移；用目标常流阈值或亚阈值 `Id–Vg` 对齐。 |
| `DeltaWGAA`, `DeltaTGAA` | 0 m | 宽、厚的工艺/CD 偏差，影响有效周长、截面积、栅电容和量子限域。 |
| `EOT_0` | 1.1 nm | 等效氧化层厚度基准，代码中还乘以网络的 C-V 输出；优先由 C–V/反型电荷提取。 |

### 几何和氧化层边界条件

| 参数 | 默认值 / 单位 | 物理含义 |
|---|---:|---|
| `TGAA` | 5 nm | 单片 nanosheet 厚度；决定截面积和量子限域。网络的 `TGAA` 归一化在此源码被固定为 -1，因此建议只在原训练厚度附近微调。 |
| `NGAA` | 3 | nanosheet 片数；决定总有效宽度与总截面积。应为正整数（当前代码支持 1–6 的显式累加逻辑）。 |
| `TFIN` | 6 nm | 用于栅源/栅漏 overlap 电容的几何尺度；用高偏置 C–V 拟合。 |
| `TOXP` | 1.07464 nm | 用于有效氧化层/电荷质心的物理氧化层厚度；与 `EOT_0` 强相关，若无 C–V 依据不要同时自由拟合。 |

### 电势、掺杂和 C–V 边界条件

| 参数 | 默认值 / 单位 | 物理含义 |
|---|---:|---|
| `NBODY`, `NSD` | 1e23, 2e26 m⁻³ | I–V 核心计算中的沟道/体区与源漏掺杂；影响体势、内建电势、阈值与短沟道效应。 |
| `PHIG` | 4.47 eV | I–V 核心计算的栅功函数；主要移动平带电压和阈值。它与 `DVTSHIFT` 高度相关，建议固定一个、拟合另一个。 |
| `NBODY_CV`, `NSD_CV`, `PHIG_CV` | 5e21, 1e26 m⁻³, 4.37 eV | 专供第二段 C–V/电荷计算的对应量。源文件在 `analog` 块中每次硬赋值；副本已移除该赋值，使其可被实例覆盖。通常应先令其与 I–V 值有物理一致性，再用 C–V 微调。 |

### 迁移率、速度饱和和输出特性

| 参数 | 默认值 / 单位 | 物理含义与数据 |
|---|---:|---|
| `U0MULT` | 1 | 对已含网络修正的迁移率退化乘性缩放；用于小幅驱动电流残差，避免与 `U0` 同时无约束拟合。 |
| `VSAT`, `VSAT1`, `VSATCV_t` | 8.5e4 m/s | 分别参与内部饱和电压、I–V 速度饱和和 C–V 速度饱和；用短沟道高 `VDS` 的 `Id–Vd`/`gm`，每次优先只调一个。 |
| `KSATIV` | 1 | 饱和电压中反型电荷项的系数；控制由线性区转入饱和区的位置。 |
| `MEXP` | 4 | `Vdseff` 平滑饱和的指数；主要影响 knee 的圆滑程度，非首选拟合量。 |
| `A1_t`, `A2_t` | 0, 0 | 非饱和/高场电流退化项；用于拟合高场曲线形状，必须加严格边界以免出现不物理曲线。 |
| `DROUT`, `PDIBL1`, `PDIBL2`, `PVAG` | 1.06, 1.3, 2e-4, 1 | 输出电导与 DIBL/栅压相关 Early 效应参数；用饱和区 `Id–Vd` 和 `gds` 拟合。 |

### 量子与寄生电容

| 参数 | 默认值 / 单位 | 物理含义 |
|---|---:|---|
| `AQMTCEN`, `BQMTCEN` | 0, 12 nm | 电荷质心几何修正的幅度和特征长度。仅在 C–V 显示明确厚度相关残差时开放。 |
| `QM0`, `PQM` | 1e-3, 0.66 | 反型电荷到电荷质心位移的幅度尺度与指数；主要校正强反型 C–V。 |
| `CFS_i`, `CFD_i`, `CGSL_i` | 2.5e-10, 2.5e-10, 1e-10 F/m | 栅源/栅漏 fringe 电容及偏置相关 overlap 电容系数。 |
| `CFS_i_CV`, `CFD_i_CV` | 1e-10, 1e-10 F/m | 第二段实际注入端口的 C–V 电荷计算所用 fringe 电容；用 `Cgs/Cgd` 或 S 参数提取。 |

## 保持为 `real` 的已初始化物理量

下列量有明确物理意义，但不适合作为常规实例拟合旋钮；它们要么是材料常数，要么当前网络已经对该物理效应做了几何依赖预测，要么在源码中没有有效接线。

| 类别 | 初始化量 | 物理含义及为何保持固定 |
|---|---|---|
| 网络范围 | `tgaamin/max`, `wgaamin/max`, `lgaamin/max`, `netmin/max` | 网络输入归一化范围；改变它会改变网络输入坐标，而不是校准器件。文件注释范围约为 `Lg=16–100 nm`、`WGAA=10–40 nm`。 |
| 材料常数 | `q`, `KboQ`, `pi`, `HBAR`, `MEL`, `mx`, `mxprime`, `md`, `mdprime`, `gprime`, `gfactor`, `EPS0`, `EPSROX`, `EPSRSUB`, `NC0SUB`, `NI0SUB`, `BG0SUB`, `EASUB` | 电荷、热电压、量子态密度、Si/SiO2 介电常数、有效质量、带隙与电子亲和势等。仅在材料体系从 Si/SiO2 改变时整体重建模型，不应局部扫参。 |
| 电荷求解常数 | `ALPHA_UFCM`, `QMFACTORCVfinal`, `QMTCENCV_i`, `UCS_t`, `DMOBCLAMP`, `DVSATCLAMP` | 统一电荷模型、量子 C–V 和数值平滑/下限控制。调节会破坏连续性、收敛性或电荷守恒。 |
| 网络已预测的迁移率 | `ETAMOB*`, `UA`, `UATHIN`, `UATSAT`, `UARTSC`, `UATNI`, `UAIR`, `EU`, `EUTHIN`, `EUPTSC`, `EUTNI`, `EUIR`, `UD`, `UDTHIN`, `UDTSAT`, `UDPTSC` | 名称对应垂直场、薄体、温度/散射相关迁移率退化。融合版实际使用 `mob_output1..5` 生成 `UA_i/EU_i/UD_i/ETAMOB_i`，因此把这些默认 `real` 改成 parameter 不会成为有效旋钮。 |
| 网络已预测的短沟道 | `DVT0`, `DVT1`, `DVT1SS`, `DSUB`, `CDSC`, `CDSCD`, `ETA0` 及其 `L/N/P/W` binning 系数 | 阈值滚降、亚阈值、DIBL 与漏致势垒效应。融合版用 `sce_output1..5` 直接参与相关公式；未接线的旧 binning 量不应开放。 |
| 未接线预留项 | `CIT`, `DVTP0`, `DVTP1`, `LPE0`, `K1RSCE` 及其 binning 系数 | 分别意图表示界面态、附加 DIBL、反短沟道与 RSCE；公式使用的是 `_i` 版本，但此版本未将这些默认量赋给 `_i`。仅改声明没有拟合作用，需先补全接线并验证。 |
| 未用/残留几何 | `HFIN`, `NFIN`, `TOXG`, `WGAANOM`, `WDIM*`, `WSSP*`, `DIM*`, `DSSP*`, `SSP*`, `TSR*`, `TDWS*`, `WSF*`, `MFE*`, `MFQ*` | 来自 BSIM-CMG/量子子模型的遗留或当前未参与主计算的尺寸/量子态参数；不应靠它们修补新节点曲线。 |
| 类型和温度 | `type`, `devsign`, `DevTemp` | 前两者定义 NMOS/PMOS 极性和端口电流符号，必须固定。`DevTemp=300 K` 是当前模型温度假设；温度扫描需要系统性加入温度依赖并重新验证，不能仅把它当任意拟合常数。 |
| 数值保护 | `DELTA_1` 与 `lexp/lln/hypsmooth/smoothminx/smoothmaxx` 的阈值 | 避免指数溢出、对数非法和不连续导数；必须保持固定。 |

## 校准约束与推荐顺序

1. 每个测试器件先填真实 `L/W/NF/TGAA/NGAA`，并保持在网络训练几何范围内；`TGAA` 的网络输入目前固定为训练值，跨度应尤其小。
2. 用 C–V 固定 `EOT_0`、`TOXP` 及必要的 `CFS_i_CV/CFD_i_CV`；不要在缺少 C–V 时同时自由拟合 `EOT_0`、`TOXP`、`QM0`、`PQM`。
3. 用长沟道低 `VDS` 拟合 `PHIG` **或** `DVTSHIFT`（二选一为主）和 `U0`；随后拟合 `XL`、`NBODY/NSD`。
4. 用短沟道高 `VDS` 拟合 `VSAT1/KSATIV`，再拟合 `DROUT/PDIBL1/PDIBL2/PVAG`；最后才考虑 `MEXP/A1_t/A2_t`。
5. 所有新增参数必须有物理范围、跨几何/偏压共同拟合，并保留独立验证集；否则会在单条曲线上过拟合且破坏网络原有的泛化能力。

## 实现备注

- 副本保留模块名 `nmos_lvt` / `pmos_lvt`，便于替换原 include 文件；同一仿真中不要同时 include 原文件与 `_2` 副本，否则模块会重名。
- 源码的截面积写为 `(WGAA + DeltaWGAA) * (TGAA + DeltaWGAA)`；按参数命名，第二个偏移很可能本应是 `DeltaTGAA`。本次只按要求开放 parameter，未改变这个原始公式，拟合时须知 `DeltaTGAA` 对面积项当前不生效。
